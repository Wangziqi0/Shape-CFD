### 各版本技术细节

**V1 文档=点云**
- 文档 $d_i$ 展开为句子级嵌入集合：$\mathcal{P}_i = \{s_{i,1}, s_{i,2}, ..., s_{i,K_i}\} \subset \mathbb{R}^{4096}$
- 拆句规则：按 `。；;` 分割，过滤 ≤5 字的短片段
- 批量 embed：Qwen3-Embedding-8B，batchSize=8，批次间 50ms 微延迟
- 预缓存：23,701 个 chunk 的句子向量离线存入 SQLite
- 代码：`ad_rank_shape.js:153-217`（buildPointClouds）

**V2 PDE 对流-扩散**
- 守恒型离散化：$C_{t+1}(i) = C_t(i) + \Delta t \cdot [\text{扩散} + \text{对流}]$
- 扩散项：$F_i^{diff} = D \sum_j W_{ij}(C_j - C_i)$（Laplacian）
- 对流项（Upwind 守恒型）：$F_i^{adv} = \sum_j W_{ij}[{\max(u_{ji},0) \cdot C_j - \max(u_{ij},0) \cdot C_i}]$
- 守恒证明：利用 $W_{ij}=W_{ji}$（无向图对称）和 $u_{ji}=-u_{ij}$（速度反对称），$\sum_i F_i^{adv} = 0$
- 对流方向：$u_{ij} = \text{uStrength} \cdot \frac{(\vec{c}_j - \vec{c}_i) \cdot \hat{q}}{|\vec{c}_j - \vec{c}_i|}$（质心连线投影到 query 方向）
- 初始条件：$C_0(i) = \exp(-2 \cdot d_{Chamfer}(P_q, P_i))$
- 时间积分：显式 Euler，浓度截断 $C \in [0, 1]$
- 代码：`ad_rank_shape.js:600-689`（PDE 核心循环）

**V3 CFL + Upwind 稳定性**
- CFL 条件：$\Delta t \leq 0.8 / \max_i(\text{degree}_i)$
- Upwind 格式保证 Perron-Frobenius 非负转移矩阵 → 唯一稳态收敛
- 收敛判据：$\max_i |C_{t+1}(i) - C_t(i)| < \epsilon = 10^{-3}$
- 最大迭代 50 步（缓存版 80 步）

**V4 Chamfer 距离**
- 对称 Chamfer：$d_{Chamfer}(A, B) = \frac{1}{2}\left[\frac{1}{|A|}\sum_{a \in A} \min_{b \in B} d(a,b) + \frac{1}{|B|}\sum_{b \in B} \min_{a \in A} d(a,b)\right]$
- 边权衰减：$W_{ij} = \exp(-\beta \cdot d_{Chamfer}(P_i, P_j))$，$\beta = 2.0$
- KNN 图：$K=3$（实验确认优于 K=5, K=7）
- 代码：`ad_rank_shape.js:98-124`（chamferDistance）

**V8 PQ-Chamfer 64×64**
- 将 4096 维切为 64 个 64 维子空间，各子空间独立计算 cosine 距离后均匀平均
- 打破高维浓度效应（Concentration of Measure）
- 消融确认：64×64 > 32×128 > 16×256（单调提升）；均匀加权 > 方差加权（-3.7%）；顺序 ≈ 打乱（Qwen3 维度无局部结构）
- 纯 PQ > PQ+混合初始场（两者解决同一问题，不正交）
- 自动维度检测：4096d → PQ-Chamfer，其他维度回退 cosine
- 代码：`ad_rank_shape.js:68-88`（pqCosineDistance）

### 各版本技术细节

**V7 扩散压力预取**
- 启发式选择：PDE 求解后，在当前候选图的边界节点中选扩散压力最大的方向扩展新候选
- 效果：NFCorpus 0.2781 (+26.7% vs cosine)
- 代码：`beir_rust_parallel.js`（v7_heuristic 方法）

**V7.1 伴随状态法（最优边界控制）**
- 最优预取准则：$i^* = \arg\max_{i \in \partial V} \left( C_i \cdot \max(0, \vec{n}_i \cdot \hat{u}_i) \right)$
- 物理释义：选择浓度高且对流场猛烈吹向图外未探索区域的节点
- 来源：Gemini 理论公式 → 实验验证
- 效果：NFCorpus 0.2802 (+27.6% vs cosine)，比启发式 +0.8%
- 代码：`beir_rust_parallel.js`（v7_adjoint 方法）

**V13 VT-Aligned 虚拟令牌**
- 核心思想：将 PQ 的 64 个子空间视为独立的"虚拟令牌"（Virtual Token）
- **反转聚合顺序**：
  - 原始 PQ-Chamfer：先跨 64 个子空间算完整向量距离 → 再做句子级 Chamfer
  - VT-Aligned：先在每个 64 维子空间内独立做句子级 Chamfer → 再跨子空间聚合
  - 即 `vtAlignedDist(qV, docCloud)`: 对每个子空间 s，query[s:s+64] 在 docCloud 中找最近句子，64 个子空间取平均
- vt_aligned 胜出 vt_unaligned（后者将 query 展开为 64 个 64d token 做全 Chamfer，噪声过大）
- 成本：零（无模型重推理，纯计算顺序变换）
- 增益：+2.5%
- 代码：`beir_rust_parallel.js`（vtAlignedDist 函数）

**UOT（非平衡最优传输）理论框架**
- Chamfer 距离 = 丢弃质量守恒约束的 Wasserstein 距离（UOT）
- W1 要求 $\sum \mu_{ij} = 1$（质量守恒），长文档 99 句废话必须一一匹配 → 灾难
- Chamfer 丢弃边缘约束，允许多对一匹配 → 每句 query 在文档中找到最近归宿
- = "硬语义注意力"（Hard Semantic Attention）
- 解释了异质文档集优势：NFCorpus（长文档）+21.5% vs SciFact（短文档）+2.6%
- Gemini 提出 → Claude 评为"整份材料最有价值论点" → 写入论文 §3.6

### 技术细节

**灵感 A：动力学解耦**
- 提出者：Gemini 第三轮 → Claude 确认
- 初始场（非对称）：$C_0(i) = \max_{v \in d_i} \cos(q, v)$（MaxSim，长文档只需一句命中）
- 图边权（对称）：$W_{ij} = \frac{d_{MaxSim}(P_i,P_j) + d_{MaxSim}(P_j,P_i)}{2}$（保守恒性）

**灵感 B：Allen-Cahn + 动态 θ**
- 提出者：Gemini 第二轮
- 反应项：$R(C) = \gamma \cdot C_i(1-C_i)(C_i - \theta(t))$
- 动态阈值：$\theta(t) = \frac{1}{|V|}\sum C_i(t)$（Volume-preserving）
- 实验结论：γ=0.5/1.0/2.0 在 top-30 均 NDCG=0.2501（不敏感）；γ=1.0 top-100 为 0.2183（-13% 灾难）
- 休眠原因：短 query 下初始分数极差 $\Delta C_0 \approx 0.1$，反应力 $\propto O(\Delta^3)$ 量级太弱

**灵感 C：(Pe, Da) 二维网格搜索**
- Buckingham π 定理：5 维 $(κ, γ, θ, D, \Delta t)$ → 2 维 $(Pe, Da)$
- $Pe = κ/D$（对流 vs 扩散），$Da = γL^2/D$（反应 vs 扩散）
- D 归一化为 1，Δt 由 CFL 锁死，θ 动态追踪 → 不是超参数

**灵感 D：对流硬截断**
- 一凡提出 VCP 浪潮算法 → Claude 对比分析
- 规则：$\|v_j - v_i\| < \epsilon \Rightarrow u_{ij} = 0$（方向不确定就不推）
- ε 自适应：k-NN 距离分布第 5 百分位，或 JL 理论下界 $\sqrt{2/d_{proj}} \approx 0.125$

**灵感 E：C 型分数分布设计**
- Claude 提出三选项（A 二值化 / B 阶梯式 / C 平滑+间隔）→ 一凡选定 C 型
- C 型三力博弈：扩散(簇内拉近) + 对流(定向推) + 反应(撕裂制造鸿沟)
- γ 旋钮：γ=0→过平滑，γ 适中→C 型，γ→∞→二值化

**LID 内蕴维度实测**
- k-NN MLE: LID ≈ 16~19
- Participation Ratio: 1316.6
- 95% variance → 3604/4096 dims（Qwen3 维度极度均匀分布）
- K=64 最优原因：非 LID 驱动，而是帕累托最优（保持流形连续性 + 打破浓度效应）
- Gemini 估计 30~80 被实测推翻

**Mix α 混合初始场**
- $C_0 = \alpha \cdot \text{MaxSim} + (1-\alpha) \cdot \text{MeanSim}$
- NFCorpus（长文档）：α=0.7 最优 +3.2%（倒 U 型）
- SciFact（短文档）：α=0.5 最优 -1.5%（单调递减）
- Gemini"α 右移"预测失败：短文档反而需要更多 MeanSim 约束（低通滤波抑制采样噪声）
- 结论：domain-dependent，不通用，纯 PQ 更稳健

### 证伪细节

**V12 PQ-Chamfer 替换 cosine（2026-03-30 核心发现）**
- stefan_full(Rust PQ-Chamfer 初始池) = 0.2615，比 cosine 粗筛更差
- Stefan Rust 探针 0.2703 也不如 cosine 探针 0.2727
- 根因：Chamfer 距离对单点 query（1 个向量 vs N 个句子）退化为普通 cosine，且多了聚合噪声
- **洞察**："cosine 不是天花板，query 单点才是"——初始检索用 cosine 粗筛是正确设计

**V5.1 归一化灾难（Gemini 第三轮交锋）**
- Min-Max 归一化将 [0.1, 0.2] 拉到 [0, 1]，空间拉伸 10×
- 等效 $\gamma_{eff} = \gamma \cdot 10^2 = 500$，Allen-Cahn 过度极化
- 修正处方（V5.2）：$\gamma_{eff} = \gamma_{base} / (C_{max} - C_{min} + \epsilon)^2$，恢复到 -0.3%

**标量势证伪**
- 标量势将各向异性的高维语义空间坍缩为各向同性的"重力漏斗"
- 只知道"谁分数高"但丢失"为什么高"的方向信息
- Gemini 提出 / Darcy 定律类比 → Claude 实验证伪 → Gemini 承认

**JL 退火假说证伪**
- V4 论文 §3.7 声称 JL 投影的方差放大 (0.016→0.088) 是有利的"流场退火"
- Gemini 严格驳斥：σ=0.088 是高斯随机投影注入的纯白噪声，不是可利用的物理信号
- 系统看似改善是因为随机噪声扮演了 Langevin 动力学的热扰动角色（类模拟退火偶然打破对称僵局）

**密度加权 Chamfer（2026-04-02 证伪）**
- 假说：密集区域的点更重要，用密度加权 Chamfer 可提升检索质量
- 实验：在多个数据集上测试密度加权变体
- 结果：NDCG -0.9% ~ -1.3%，均匀加权最优
- 根因：密集区反映的是嵌入模型的冗余分布模式，不是语义重要性信号。min 操作天然选最近邻，密度加权是多余干扰

**PQ 码重建向量（2026-04-02 证伪）**
- 假说：用 PQ 量化码重建近似原始向量，再计算 Chamfer 距离
- 结果：NDCG 暴跌 34%
- 根因：PQ 量化在每个子空间内有 ~10% 误差，重建后 64 个子空间误差叠加。Chamfer 的 min 操作进一步放大量化误差（选最近邻对距离精度极度敏感），导致排序崩溃
- 教训：PQ 码适合做子空间独立距离计算（PQ-Chamfer +28%），不适合重建完整向量后再计算距离
