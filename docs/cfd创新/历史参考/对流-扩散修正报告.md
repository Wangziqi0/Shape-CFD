## 二、问题 1：对流公式方向性

### 2.1 原公式（指令原文）

```
F_i^adv = -Σ max(u_ij, 0) × W_ij × (C_j - C_i)
```

### 2.2 验证：两节点场景

设 `u_01 > 0`（风从节点 0 吹向 1），`C_0 = 0.9` > `C_1 = 0.2`：

| | 节点 0（上游） | 节点 1（下游） |
|---|---|---|
| **原公式** | +0.21（增大！❌） | 0（不变！❌） |
| **守恒型** | -0.27（流出 ✅） | +0.27（流入 ✅） |

**原公式是 downwind scheme**：上游节点浓度反而增大，物理上等价于"负扩散"（逆平滑）。
被 `clamp(0,1)` 掩盖了数值不稳定性。

### 2.3 修正后公式（守恒型 upwind）

```
F_i^adv = Σ_j W_ij × [max(u_ji, 0)·C_j - max(u_ij, 0)·C_i]
```

**物理含义**：节点 i 的浓度变化 = (从各上游 j 流入的浓度) - (向各下游 j 流出的浓度)

**守恒证明**：对每条边 {i,j}，设 u_ij > 0：
- 节点 i 贡献: -u_ij × C_i（流出）
- 节点 j 贡献: +u_ij × C_i（流入）
- 和 = 0 ✅

Gemini 预测的结论经验证完全正确：守恒型 upwind 下 **Σ F_adv = 0**。

## 三、问题 2：雷诺数退化

### 3.1 根本原因

在对称 KNN 图上，扩散项全局和恒等于零（通量守恒）：

```
Σ_i F_diff_i = D × Σ_{(i,j)} W_ij × [(C_j-C_i) + (C_i-C_j)] = 0
```

这使得指令原文的 `Re = |Σ 对流| / |Σ 扩散| = 非零/0 → undefined`。

> **物理本质**（Gemini 论述）：
> 标量浓度传输中，此无量纲数应称为 **佩克莱数（Péclet Number）** 而非雷诺数。
> 封闭对称图上扩散满足绝对质量守恒，代数和必为 0。

### 3.2 全部可选方案

| 方案 | 公式 | Re 值 | 优缺点 |
|------|------|-------|--------|
| **A** | \|Σ F_adv\| / \|Σ F_diff\| | 恒为 0 | ❌ 分母恒零，无意义 |
| **B** ⭐ | Σ\|F_adv_i\| / Σ\|F_diff_i\| | 1.22 | ✅ 稳健，测量"活动强度" |
| **C** | mean(\|F_adv\|) / mean(\|F_diff\|) | = B | ≡ 等价于 B（1/N 约去） |
| **D** | mean(\|F_adv_i\|/\|F_diff_i\|) | 12.5 | ❌ 奇异点爆炸（F_diff≈0 处 Re_i→∞） |
| **E** ⭐ | mean(\|u_ij\|) / D | 1.09 | ✅ 静态常数，最符合 CFD 物理 |

### 3.3 方案 D 奇异点示例

```
Re[0]=10.0  Re[1]=0.8  Re[2]=0.7  Re[3]=0.7
Re[4]=50.0  Re[5]=0.7  Re[6]=60.0 ← 尖峰！
mean = 12.5（被少数尖峰拉高 5 倍）
median = 0.8（中位数更稳健但仍不如 B）
```

### 3.4 方案 E 理论基础

Gemini 提出的**本征图佩克莱数**，灵感来自 CFD 单边佩克莱数：

```
单边: Pe_ij = |u_ij| × W_ij / (D × W_ij) = |u_ij| / D
全图: Pe = mean(|u_ij|) / D
```

W_ij 在分子分母约去，得到极其纯粹的结果。**不依赖浓度场 C，建图时计算一次**。

与参考论文 §9 **Advection Augmented CNN (NeurIPS 2024)** 中的 Péclet 概念一致。
