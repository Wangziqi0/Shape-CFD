# AD-Rank：基于对流-扩散方程的通用向量检索增强排序算法

> **版本**: v4.0 (2026-03-28)
> **定位**: 通用词向量检索后排序算法，不绑定任何特定领域
> **状态**: PQ-Chamfer 64×64 集成完成，NFCorpus NDCG@10=0.2735 (+24.6% vs cosine)

---

## 1. 研究动机

### 1.1 问题

现有 RAG (Retrieval-Augmented Generation) 系统的检索环节依赖 cosine similarity 直接排序：

```
query → embedding → cosine(query, doc) → Top-K → LLM
```

**局限性**：
1. **孤立打分**：每个文档独立评分，忽略文档间的语义关联
2. **无方向性**：cosine 是对称度量，无法表达"查询方向"

### 1.2 核心思想

借鉴 CFD（计算流体力学）中的**对流-扩散方程**：

- **扩散项**：文档间的语义传播（高相关度向邻居扩散，发现隐含相关文档）
- **对流项**：query 方向性（根据 query 的"语义风向"，让排序有方向偏好）

```
C_{t+1}(i) = C_t(i) + Δt × [扩散项 + 对流项]
```

### 1.3 与已有工作的差异

| 方法 | 扩散 | 对流 | 方向性 |
|:---|:---:|:---:|:---:|
| Cosine Similarity | ❌ | ❌ | ❌ |
| Diffusion-Aided RAG (ACL 2025) | ✅ | ❌ | ❌ |
| PageRank / HITS | 类似 | ❌ | ❌ |
| **AD-Rank (本工作)** | **✅** | **✅** | **✅** |

**核心创新**：**对流项**。这是 AD-Rank 相对于纯扩散方法的独有贡献。

## 2. 数学框架

### 2.1 图构建

给定 N 个候选文档的 embedding 向量 $\{v_1, ..., v_N\} \in \mathbb{R}^d$：

1. **余弦相似度矩阵**：$S_{ij} = \cos(v_i, v_j)$
2. **KNN 稀疏图**：每个节点保留 top-k 近邻
3. **边权重**：$W_{ij} = \exp(-\beta \cdot (1 - S_{ij}))$，$\beta = 2.0$
4. **对称化**：若 $i \to j$ 存在而 $j \to i$ 不存在，补充反向边

### 2.2 初始浓度场

$$C_0(i) = \cos(q, v_i)$$

每个文档的初始"浓度"就是它与 query 的 cosine similarity。

### 2.3 对流方向

对每条边 $(i, j)$，计算对流系数：

$$u_{ij} = \langle \frac{v_j - v_i}{\|v_j - v_i\|}, \frac{q}{\|q\|} \rangle \cdot u_{\text{strength}}$$

**物理含义**：边方向与 query 方向的一致程度。$u_{ij} > 0$ 表示"风"沿 $i \to j$ 方向吹。

**反对称性**：$u_{ji} = -u_{ij}$（边方向反转）。

### 2.4 迭代方程（守恒型 upwind 格式）

$$C_{t+1}(i) = C_t(i) + \Delta t \cdot \left[ \underbrace{D \sum_j W_{ij} (C_j - C_i)}_{\text{扩散项}} + \underbrace{\sum_j W_{ij} [\max(u_{ji}, 0) \cdot C_j - \max(u_{ij}, 0) \cdot C_i]}_{\text{守恒型 upwind 对流项}} \right]$$

> **⚠️ 关键修正**：原始设计使用了 downwind scheme（$-u_{ij} \cdot W_{ij} \cdot (C_j - C_i)$），
> 经 Gemini 3.1 Pro 审查发现物理方向反了——上游节点浓度会凭空增大。
> 修正为守恒型 upwind 格式后，全局通量守恒（$\sum_i F_i^{adv} = 0$）。

### 2.5 收敛条件

$$\max_i |C_{t+1}(i) - C_t(i)| < \epsilon$$

采用 $\epsilon = 10^{-3}$（检索排序不需要高精度），配合自适应时间步长。

### 2.6 双指标体系

| 指标 | 公式 | 含义 | 特性 |
|:---|:---|:---|:---|
| **动态 Reynolds 数** | $Re = \frac{\sum_i |F_i^{adv}|}{\sum_i |F_i^{diff}|}$ | 对流活动/扩散活动的比值 | 随迭代变化 |
| **结构 Péclet 数** | $Pe = \frac{\text{mean}(|u_{ij}|)}{D}$ | 图结构的对流/扩散能力比 | 静态常数，零运行时开销 |

> **为何需要两个指标**：
> - 在对称图上，扩散全局和恒为零（质量守恒），使得传统 $Re = |\sum F_{adv}| / |\sum F_{diff}|$ 退化为 $\frac{\text{非零}}{0}$
> - 方案 B（逐节点取绝对值后求和）解决了分母问题
> - 方案 E（结构 Péclet 数）提供了不依赖浓度场 $C$ 的静态特征量

### 2.6.5 自适应参数 (Agent 9)

为消除手动调参，引入基于图结构的自适应超参数机制：

#### Fiedler Value 自适应 D

计算拉普拉斯矩阵 $L$ 的第二小特征值 $\lambda_2$（Fiedler value，代数连通度）：

$$D_{\text{adaptive}} = \frac{D_{\text{base}}}{1 + \lambda_2}$$

- $\lambda_2$ 大（图紧密）→ D 减小，防过度平滑
- $\lambda_2$ 小（图分散）→ D 增大，增强信号传播

#### 跨域检测自适应 uStrength

通过文档间相似度方差 $\sigma^2$ 线性插值：$u_{\text{adaptive}} = 0.1 + 0.2 \cdot \min(\sigma^2/0.04, 1)$

#### 验证结果

| Query | Fiedler λ₂ | D 调整 | U 调整 | 跨域 |
|:---|:---:|:---|:---|:---:|
| 劳动合同解除赔偿 | 0.374 | 0.15→0.109 | 0.1→0.151 | 否 |
| 醉驾量刑标准 | 0.081 | 0.15→0.139 | 0.1→0.158 | 否 |

物理逻辑正确："劳动合同" λ₂=0.37（图紧密）→ D 显著减小；"醉驾量刑" λ₂=0.08（图分散）→ D 几乎不变。

### 2.7 流场拓扑分析

对稳态浓度场计算每个节点的散度：

$$\text{div}(i) = \sum_j u_{ij} \cdot W_{ij} \cdot C_j$$

| 类型 | 条件 | 物理含义 | 检索含义 |
|:---|:---|:---|:---|
| **汇聚点** | $\text{div}(i) < -\theta$ | 流体汇聚 | 多个语义方向指向此文档 |
| **发散点** | $\text{div}(i) > \theta$ | 流体发散 | 语义从此文档向外辐射 |
| **停滞点** | $|dC/dt| \approx 0$ | 流场静止 | 扩散和对流平衡 |
| **流动** | 其他 | 正常流动 | 正常语义传播 |

**核心思想**：将 4096d cosine 距离切为 64×64 子空间独立计算后均匀平均，缓解高维浓度效应。

#### BEIR Benchmark (NFCorpus, 323 queries, 32 workers)

| Method | NDCG@10 | vs cosine |
|:-------|:-:|:-:|
| cosine baseline | 0.2195 | — |
| AD-Rank v2 | 0.2184 | -0.5% |
| Shape-CFD V4 (全维 Chamfer) | 0.2508 | +14.3% |
| BM25 | 0.2573 | +17.2% |
| Mix α=0.7 | 0.2588 | +17.9% |

| Shape-CFD JL128 (随机投影) | 0.2316 | +5.5% |

#### 消融实验

- **切分数量**: 16×256 (0.2650) → 32×128 (0.2684) → **64×64 (0.2729) 🏆**
- **加权方式**: 均匀 (0.2684) > 方差加权 (0.2586, -3.7%)
- **维度顺序**: 顺序 (0.2684) ≈ 打乱 (0.2681)
- **PQ + 混合初始场**: 不正交，叠加后下降 (-13%)

#### 🔒 最终确认的最优配置

```
距离函数:   pqCosineDistance (64×64, 均匀加权, 顺序切分)
初始场:     纯 Chamfer (α=1.0, 不混合)
KNN:        k=3
扩散系数:   D=0.15
对流强度:   uStrength=0.3
最大迭代:   50, 收敛阈值 1e-3
```

**生产集成**: `ad_rank_shape.js` 中 `chamferDistance()` 自动检测 4096d → PQ，其他维度回退原始 cosine。零 API 改动，向后兼容。

## 6. 关键技术修正记录

### 6.1 对流公式修正（Gemini 3.1 Pro 审查发现）

| | 原始公式 (downwind) | 修正后 (upwind) |
|:---|:---|:---|
| 公式 | $-u_{ij} W_{ij} (C_j - C_i)$ | $W_{ij}[\max(u_{ji},0)C_j - \max(u_{ij},0)C_i]$ |
| 方向 | ❌ 上游增大 | ✅ 上游流出、下游流入 |
| 守恒 | ❌ $\sum F \neq 0$ | ✅ $\sum F = 0$ |
| 稳定性 | 需 clamp 掩盖 | 自然有界 |

### 6.2 雷诺数修正

原始方案（$Re = |\sum F_{adv}| / |\sum F_{diff}|$）分母恒为零（对称图扩散守恒）。
替换为双指标体系：动态 Re（方案 B）+ 结构 Pe（方案 E）。

## 7. 理论延伸（v2 路线图）

### 7.1 分块自适应对流 (BAA, Block-Adaptive Advection)

将 4096 维 embedding 分成 B 个块，每块独立：
- 独立建 KNN 图
- 独立计算对流方向
- 跨块融合：$C_{final} = \sum_b w_b \cdot C_b + \lambda \cdot \text{cross\_block\_term}$

**灵感来源**：GPU 量化中的分块策略（GPTQ/AWQ），不同维度子空间可能携带不同语义方向。

### 7.2 自适应 Reynolds 控制

**已部分实现 (Agent 9)**：通过 Fiedler value 自适应调 D + 跨域检测自适应调 uStrength。

待扩展：
- 根据 Re 反馈回路动态调参（Re > 2 减小 u，Re < 0.5 增大 u）
- 多轮迭代中渐进调参

### 7.3 谱加速

利用图拉普拉斯 $L$ 的特征分解加速扩散：
- $K = \exp(-\beta L)$ 是解析解
- 用 Lanczos 算法近似前 k 个特征向量
- 将 $O(T \cdot N^2)$ 迭代降为 $O(k \cdot N)$ 矩阵乘
