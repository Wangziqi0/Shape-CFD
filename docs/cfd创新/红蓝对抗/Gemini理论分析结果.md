# Gemini 理论分析结果 (2026-03-25)

> Gemini 对 AD-Rank 9 个开放问题的数学分析
> 模型：Gemini 标准模型（deepthink 高负载不可用）

## 关键数学结论

### 1. 维度诅咒 — 对流弱的根因已确认
- d=4096 时，随机向量 dot product 标准差 σ = 1/√4096 ≈ 0.0156
- 实验 mean(|u_ij|) ~ 0.01 与理论完全吻合
- **解法**：
  - **局部 PCA 降维**：对 30 候选做 SVD 投影到前 k=32 主成分
  - **温度缩放**：u_ij = sgn(x)×|x|^α (α<1) 非线性放大

### 2. BAA 分块 — 信号增强 √B 倍
- 分 B=8 块后，每块 512 维，信号增强 √8 ≈ 2.83 倍
- 不同注意力头确实编码不同语义（块状协方差结构）
- 等价于**多视图图学习**
- **权重 w_b**：按 Query-Block 对齐度动态确定（最优解）

### 3. 反对称性 — 欧氏空间中严格成立
- v_j - v_i = -(v_i - v_j)，点积必然反号
- 放弃反对称 → 质量不守恒 → 需 Sinkhorn 归一化
- 非对称语义距离有理论支持（KL 散度），但增加复杂度

### 4. 扩散主导 — 不是问题
- 对流=各向异性"向导"，防止纯扩散的过度平滑
- 16% 的排序差异 = 核心亮点，不是劣势
- **足够发论文**

### 5. 谱加速 — N=30 矩阵指数可能更快
- N³ = 27,000 次运算，纳秒级
- 但对流使矩阵非对称 → 复数特征值 → 不稳定
- 混合方案有 Trotter-Suzuki 截断误差 → 不推荐

### 6. 跨法域效果好 — query 悬在聚类之间
- 跨域 query 位于两个聚类之间的稀疏地带，梯度方向明确
- 单一法域内，候选全挤一起，梯度退化为噪声
- **可利用**：用图密度/特征值分布自动调 uStrength

### 7. 非欧几何 — 信号增强但工程成本高
- 双曲空间边缘方向更锐利
- 需对数映射、切空间计算
- **性价比低**，不推荐落地

### 8. 通用性 — Fiedler Value 自动适配
- 最大隐含假设：聚类假设
- 稀疏分布 → 增大 D；紧密聚集 → 减小 D
- **D = f(1/λ₂)**：拉普拉斯第二小特征值自动决定扩散系数

### 9. 向量→形状 — 核心突破路径
- **文献支持**：Gaussian Embeddings (2014)、Box Embeddings、Poincaré Embeddings
- **工具**：SPH（无网格粒子法）、Wasserstein 距离、Chamfer 距离
- **理论障碍**：高维体积坍缩 → 必须用距离度量而非密度重叠
- **MVP 实验**：
  1. 候选法条按句子拆分 → 每个文档变成点云
  2. KNN 边权重用 Chamfer 距离倒数替代 cosine
  3. 对流方向 = 文档点云质心 → query 点云质心
  4. 在新图上跑现有 AD-Rank 代码
