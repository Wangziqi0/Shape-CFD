## 1. V7 伴随状态法最优预取准则

**Gemini 给出的公式：**

$$i^* = \arg\max_{i \in \partial V} \Big( C_i \cdot \max(0, \vec{n}_i \cdot \vec{u}_i) \Big)$$

物理释义：节点不仅浓度高，且对流场正猛烈吹向图外未探索区域。

**Antigravity 评估：✅ 正确且可实验验证**

- 当前启发式（高浓度+低度数）是这个公式的粗糙近似
- $\vec{n}_i$ (向外法向) ≈ 低度数节点的"边界方向"
- $\vec{u}_i$ (对流场) = 我们已经计算的对流系数
- **可以用这个公式替换启发式，看能否超过当前 V7 的 +1.9%**

## 2. V10 ≠ PPR（三道鸿沟）

| 维度 | PPR | Shape-CFD V10 | Antigravity 评估 |
|:--|:--|:--|:--|
| 驱动场矢量性 | 各向同性扩散 | 各向异性对流 | ✅ **成立** — query 驱动的有向流场是核心创新 |
| 系统线性 | 线性马尔可夫 | 非线性 Allen-Cahn | ⚠️ **问题** — Allen-Cahn 在我们系统中已被证伪（O(Δ³) 休眠），不能作为区分 PPR 的论据 |
| PDE 分类 | 热传导方程 | 对流-扩散方程 | ✅ **成立** — Upwind 格式的对流项是本质区别 |

**Antigravity 修正：V10 与 PPR 的真正区别是 2 道，不是 3 道。**
- ✅ 对流项（方向性）
- ✅ PDE 分类（对流-扩散 vs 纯扩散）
- ❌ 非线性反应项（Allen-Cahn 休眠，不能用）

如果论文审稿人问"这和 PPR 有什么区别"，只打前两张牌。第三张牌在当前参数下站不住。

## 3. K=64 理论最优 = 局部内蕴维度 (LID)

**Gemini 论点：**
- K→1 时 cosine 退化为符号比较（汉明距离）→ 拓扑坍缩 ✅ 正确
- K→∞ 时高维浓度效应 → 溜冰场效应 ✅ 正确
- K=64 恰好 = Qwen3 的 LID ⚠️ **需要实测**

**Antigravity 评估：方向正确，但 LID=30~80 是估计值，未在 Qwen3-8B 上实测**

可以用简单方法估算 LID：
- 对 NFCorpus 语料库向量做 PCA，看方差解释率在哪个维度饱和
- 或用 k-NN 估计法 (Levina & Bickel 2005)

**Action: 在 NFCorpus 上实测 Qwen3 embedding 的 LID**

### 实测结果 (2026-03-28)

```
k-NN MLE LID Estimator (500 samples):
  k=5:  LID = 18.9
  k=10: LID = 15.8
  k=20: LID = 15.8
  k=50: LID = 17.3

Participation Ratio: 1316.6
95% variance → 3604 dims (of 4096)
Top 64 dims → 15.0% variance
```

**Gemini 的 LID=30~80 估计被推翻。实测 LID ≈ 16~19。**

但 K=64 仍然是最优——原因不是 64=LID，而是：
1. m=64 足够大，保持了流形连续性（cosine 不退化为符号比较）
2. m=64 足够小，方差放大 64 倍打破了浓度效应
3. Qwen3 信息极度均匀分布（95% 方差需要 3604 维），所以方差加权无效

**修正后的理论解释**：K=64 是"距离方差放大"与"流形连续性保持"的帕累托最优点，与 LID 无直接关系。

## 4. Chamfer = 非平衡最优传输 (UOT) ← 最强论点

**Gemini 论点：**
- W1 要求质量守恒 → 长文档的 99 句废话必须匹配 → 灾难性噪声惩罚
- Chamfer 丢弃守恒约束 → 允许多对一 → 只关心"每句 query 在文档中找到归宿"
- = 无边缘约束的非平衡最优传输 (UOT without Marginal Constraints)

**Antigravity 评估：✅ 完全正确，这是整份材料中最有价值的论点**

- 直接解释了为什么 Chamfer > cosine（cosine 是 1句→1句的退化情况）
- 直接解释了为什么短文档上 Chamfer 优势小（SciFact），长文档优势大（NFCorpus）
- "Hard Semantic Attention" 的表述精确且有画面感
